Продолжаем учиться верстать.
На этот раз вот такой простенький макет: https://www.figma.com/file/ZRT1lTxs8KQtlbvl33dMRb/alivio-landing-page-for-figma?type=design&node-id=0-1
Верстка как обычно - резиновая, имею ввиду, что layout меняется в зависимости от ширины экрана, не привязываясь к определенным размерам.
В этом небольшом лендинге столкнулся с необходимостью подключать шрифты из сторонних ресурсов, в гугл фонтс их банально не было, как в предыдущих проектах.
Найдя необходымые шрифты, я подключил папку со шрифтами (fonts) в html документ.
Из JavaScript - самостоятельно реализовал меню-бургер для мобильной версии сайта.
На этом этапе я еще не изучал JS, и даже простой код в 5 строчек по видеоурокам в ютуб, оказался для меня жутко тяжелым.
Тем не менее, реализовав задуманное, было принято решение - изучать JS, и параллельно подтягивать css.
